# ObjectC_protocol
Object C 添增協定，以實現類別的多重繼承功能

協定事實上是一組方法列表，不依賴於特定類別，使用協定可以使不同類別共用相同訊息。
很類似後來 swift 的 delegate 委派功能。

宣告協定
 
                   @protocol 協定名稱
                   
                   @end
                   
聲明方法

                   @protocol 協定名稱
                   
                   @required 
                   實現協定的必須方法
                   
                   @end
